# 🚕 Taxi App - Полноценное приложение такси

Современное веб-приложение такси с фронтендом на React/TypeScript и бэкендом на Node.js/Express с базой данных Supabase.

## 🛠 Технологический стек

### Frontend
- React 19 + TypeScript
- Vite (сборщик)
- TailwindCSS (стилизация)
- WebSocket для real-time обновлений

### Backend
- Node.js + Express.js + TypeScript
- Supabase (PostgreSQL) - база данных
- Socket.IO - WebSocket сервер
- JWT аутентификация
- Express Rate Limiting
- CORS, Helmet для безопасности

## 📋 Требования

- Node.js 18+ и npm
- Аккаунт Supabase (бесплатный)

## 🚀 Инструкция по запуску

### Шаг 1: Настройка Supabase

1. Перейдите на [supabase.com](https://supabase.com) и создайте аккаунт
2. Создайте новый проект
3. Скопируйте URL проекта и ключи API
4. В SQL Editor выполните скрипт из файла `backend/supabase-schema.sql`

### Шаг 2: Настройка бэкенда

```bash
# Перейдите в папку бэкенда
cd backend

# Установите зависимости
npm install

# Скопируйте файл конфигурации
cp .env.example .env

# Отредактируйте .env файл
nano .env
```

Заполните `.env` файл:
```env
# Supabase Configuration
SUPABASE_URL=https://ваш-проект.supabase.co
SUPABASE_ANON_KEY=ваш_anon_key
SUPABASE_SERVICE_ROLE_KEY=ваш_service_role_key

# Server Configuration
PORT=3001
NODE_ENV=development

# JWT Configuration
JWT_SECRET=ваш_секретный_ключ_jwt
JWT_EXPIRES_IN=24h

# CORS Configuration
FRONTEND_URL=http://localhost:5173
```

```bash
# Запустите бэкенд в режиме разработки
npm run dev
```

Бэкенд запустится на `http://localhost:3001`

### Шаг 3: Настройка фронтенда

```bash
# Откройте новый терминал и перейдите в корневую папку
cd ..

# Установите зависимости фронтенда
npm install

# Запустите фронтенд
npm run dev
```

Фронтенд запустится на `http://localhost:5173`

## 🎯 Использование приложения

### Роли пользователей

1. **CLIENT (Клиент)** - может заказывать поездки
2. **DRIVER (Водитель)** - может принимать заказы
3. **ADMIN (Администратор)** - может управлять системой

### Функционал

#### Для клиентов:
- ✅ Регистрация/вход по номеру телефона
- ✅ Заказ поездки (указание маршрута, типа поездки)
- ✅ Отслеживание водителя в реальном времени
- ✅ Чат с водителем
- ✅ История поездок
- ✅ Управление профилем и платежными картами
- ✅ Кошелек

#### Для водителей:
- ✅ Панель управления с заработком
- ✅ Получение новых заказов
- ✅ Обновление статуса (онлайн/офлайн)
- ✅ Управление текущей поездкой
- ✅ Чат с клиентом
- ✅ Статистика заработка

#### Для администраторов:
- ✅ Просмотр всех водителей
- ✅ Мониторинг поездок
- ✅ Статистика системы

## 🔄 Real-time функции

Приложение поддерживает обновления в реальном времени:
- Уведомления о назначении водителя
- Обновления локации водителя
- Сообщения в чате
- Изменения статуса поездки

## 🐛 Устранение проблем

### Проблемы с подключением к Supabase

1. Проверьте правильность URL и ключей в `.env`
2. Убедитесь, что выполнили SQL схему
3. Проверьте правила RLS в Supabase

### Ошибки CORS

Убедитесь, что `FRONTEND_URL` в `.env` бэкенда соответствует адресу фронтенда

### Проблемы с WebSocket

Проверьте, что порт 3001 не заблокирован фаерволом

## 🚀 Деплой в production

### Бэкенд (например, на Railway)
1. Создайте аккаунт на Railway
2. Подключите GitHub репозиторий
3. Настройте переменные окружения
4. Деплойте проект

### Фронтенд (например, на Vercel)
1. Создайте аккаунт на Vercel
2. Подключите GitHub репозиторий
3. Измените `API_BASE_URL` в `api.ts` на адрес production бэкенда
4. Деплойте проект
